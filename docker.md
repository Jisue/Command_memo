<br>

## ⚙️ 명령어

### 이미지 실행/저장
```
> docker run -it '이미지 이름'
```

### 이미지 확인
```
> docker images
```

### 컨테이너 확인
```
> docker ps     // 실행중인 컨테이너
> docker ps -a  // 모든 컨테이너
```
